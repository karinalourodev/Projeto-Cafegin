#  Cafegin — Site Institucional

<img src="assets/logo-cafegin.png" alt="Logo da Cafegin" width="200">

## 📖 Sobre
Site institucional desenvolvido para a **Cafegin**, cafeteria que valoriza tradição e frescor.  
O projeto tem como objetivo fortalecer a presença digital da marca e aproximar clientes.

## 🚀 Tecnologias
- HTML5  
- CSS3  
- JavaScript  
- Google Fonts  
- Integração com WhatsApp  

## 🎨 Funcionalidades
- Menu responsivo e interativo  
- Scroll suave entre seções  
- Seções institucionais: Home, Sobre, Produtos e Contato  
- Cards de produtos com botão de compra  
- Identidade visual personalizada  

## 📱 Responsividade
Layout adaptado para **desktop, tablet e mobile**, garantindo navegação intuitiva em qualquer dispositivo.
